Menghitung LUAS dan KELILING lingkaran

Deklarasi

    INPUT r = "integer"

    INPUT phi = 22/7 or 3.14

Deskripsi

    READ AND SAVE r

    READ AND SAVE phi

    CALCULATE LUAS = phi * r * r 

    CALCULATE KELILING = 2 * phi * r

End

    WRITE ("LUAS LINGKARAN=" LUAS)

    WRITE ("KELILING LINGKARAN=" KELILING)
